# Проект сапер на C

## Функционал проекта

- Классические правила сапёра
- Открытие ячеекЮ возможность ставить флажки
- Отображение сколько мин рядом
- Безопасный 1 ход
- Умное открытие (дабл клик)
- Спец ячейки
- Система способностй
- Разные уровни сложности

## Структура проекта

minesweeper_project/
|- src/ - исходный код
|- docs/ - документация
|- build/ - скомпилированные файлы
|- read.md - этот файл

## Модули проекта
main.c - главное меню
game.c - основная логика игры
field.c - игровое поле
ui.c - интерфейс
